WsdlUI

A program for generating web service requests

View project on GitHub

Welcome to WsdlUI.

WsdlUI is a program for quickly viewing and testing web services generated from a .wsdl file. The program is designed to be fast and responsive implementing a small number of features needed for testing web services.
The project is free, open source (GPL v3) and being actively developed, if you have any feedback or feature requests please get in touch.

Requirements Windows

Windows XP SP2 (and above Windows 7, Windows 8)
.NET v3.5

Requirements Linux

Mono 2.10.X or higher, Mono WinForms library.

sudo apt-get install libmono-winforms2.0-cil
Install mono on Debian based systems (Ubuntu, Mint)
sudo yum install mono-winforms.x86_64
Install mono on RedHat based systems (Fedora)

Installation Windows

Download the zip file, unzip then run the msi file its as simple as that.
Alternatively to run without installation, simply unzip and run WsdlUI.exe from the release folder.

Installation Linux

Download the tar file, extract then run mono WsdlUI.exe from the release folder.
WsdlUI uses the Inconsolata font if it is available, otherwise download and install it at Inconsolata.
