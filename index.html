Welcome to WsdlUI.

WsdlUI is a program for quickly viewing and testing web services generated from a .wsdl file. The program is designed to be fast and responsive implementing a small number of features needed for testing web services.
The project is free, open source (GPL v3) and being actively developed, if you have any feedback or feature requests please get in touch.
